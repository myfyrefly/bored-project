# bored-project

![Never Bored Homepage](/images/neverBored.jpg "Never Bored Homepage")

Never Bored is an app that suggests activities to do in your area based on user criteria.

# User Story

AS a person who gets tired and bored of the Facebook realm,
I WANT to change things up and find new things to try in areas near me
SO THAT I can get rid of my boredom and discover new places at the same time.

# Libraries/Technologies Used

- Materialize CSS
- Animate CSS
- Bored API: https://www.boredapi.com/
- Google Maps
